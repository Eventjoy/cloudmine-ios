Detailed Description

This is a class to represent an ACL object in CloudMine's data store. CMACLs can be added to CMObjects to share CMObjects with other users. The users specified in the members property are granted the permissions specified in the permissions property. ACLs can only be added to user-level objects, and only by the owner of those objects.
